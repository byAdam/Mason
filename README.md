# Mason
![Logo](media/misc/logo.png)

Mason is a Minecraft Addon for Bedrock 1.12 that adds many useful survival orientated tools! 

**Note**: This addon is currently in beta, as it Bedrock 1.12 so it will be prone to bugs

## Download: V0.1
